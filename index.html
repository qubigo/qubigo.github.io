Experience the art of 3D Strategy with Qubigo

Qubigo is a strategic game designed for 2 players, where the goal is to create a row of 3 pieces with a shared characteristic.

Players must develop a 3D strategy while preventing their friends from scoring points. With just 26 moves to victory, Qubigo is a game that requires a sharp mind and calculated risk-taking.

Game rules

The game is played on a cube with 27 positions. Players take turns placing a piecce in one of these positions. The pieces come in 3 shapes, 3 colors, and 3 structures.

The objective of the game is to create a series of 3 pieces with one or more common characteristics in a row, in any direction.

To make a move

  1. select piece
  2. select position
  3. press ok button

If a player creates a series of 3 pieces with a common characteristic in a row, they earn a point.

3 x yellow = 1 point
3 x cube = 1 point
3 x solid = 1 point
3 x open = 1 point
3 x striped = 1 point

If a player creates a series of three pieces with 2 common characteristics in a row, they earn 2 points.

3 x solid cube = 2 points
3 x green striped = 2 points
3 x open pyramid = 2 points
3 x yellow cube = 2 points

The game continues until all positions on the cube are filled, at which point the player with the most points wins.
