Game rules

The game is played on a cube with 27 positions. Players take turns placing a stone in one of these positions. The pieces come in three shapes, three colors, and three structures.

The objective of the game is to create a series of three pieces with a common characteristic in a row, in any direction.

To make a move

  1. select piece
  2. select position
  3. press ok button

If a player creates a series of three pieces with a common characteristic in a row, they earn a point.

3 x yellow = 1 point
3 x cube = 1 point
3 x solid = 1 point
3 x open = 1 point
3 x stripe = 1 point

If a player creates a series of three pieces with two common characteristics in a row, they earn two points.

3 x solid cube = 2 points
3 x green stripe = 2 points
3 x open pyramid = 2 points
3 x yellow cube = 2 points

The game continues until all positions on the cube are filled, at which point the player with the most points wins.
